# Blood donation (CRUD)

A basic blood donation history database. Implemented a simple CRUD system; user can create, read, update, and delete data (blood donation history) accordingly. The database shows data regarding: donation ID, donation date, donated blood type, and if user donated anonymously or not. Database used is supabase.

### Built with
- [Vite](https://vitejs.dev/)
- [React.js](https://react.dev/)
- [Redact Query](https://tanstack.com/query/v3/)
- [React Hook Form](https://react-hook-form.com/)
- [Supabase](https://supabase.com/)
- [Tailwind CSS](https://tailwindcss.com/)
